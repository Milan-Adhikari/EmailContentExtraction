To use the program:
Steps:
1. Go to the Google Account that you want to read the emails from.
2. Turn on the 2-factor authentication
3. Generate an App password and use that App password in the script.
# Note: App password generated are deleted after one use in google account, so please
save the app password in some place because you will need to feed it to the script 
everytime you run the script. You dont need to feed the password of the google account,
but the app password generated.
